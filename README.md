# Maven Finance

Maven Finance is a decentralized, non-custodial, and community-governed financial platform that enables users to earn yield on their assets and access liquidity through peer-to-peer lending and multi-asset-backed loans. The platform is designed to provide financial freedom and transparency while integrating decentralized governance and pricing oracles.

## Documentation

Comprehensive documentation is available to help users, developers, and contributors understand and interact with the Maven Finance ecosystem. It includes technical details, smart contract explanations, governance mechanisms, and integration guides.
